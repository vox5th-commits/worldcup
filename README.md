# 2026 월드콕 승부예축 (worldcup)

이 페이지는 기존 world 페이지를 기반으로 한 친구 모임용 승부예축 토토 페이지입니다.

## 사용법
1. GitHub Pages 설정 (Settings > Pages > Source: Deploy from a branch > main / (root) )
2. index.html 파일 업로드 및 수정
3. group-photo.jpg 와 배경음악(worldcup-bgm.mp3) 파일 추가 시 사용 가능

## 요청사항 반영
- 2경기 + 우승국 예축으로 조정 예정
- 전체 한꾌번에 마감 기능 구현 예정

상세 사항은 요청 주세요!
